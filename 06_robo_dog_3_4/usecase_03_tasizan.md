# ユースケース名

足し算

# ユースケース概要

2 つの整数の足し算をする

# アクター

ユーザー

# 事前条件

- メニュー画面が表示されている

# 事後条件

- 足し算結果を表示する

# 基本フロー

1. ユーザーは、RoboDog システムメニューの「２つの整数の足し算」を選択する
2. システムは、「２つの整数の足し算」
3. システムは、「名前の変更画面」を表示する
4. ユーザーは、「名前の変更画面」に新しい名前を入力する
5. システムは、新しい名前を、RoboDog に登録する
6. システムは、名前の変更結果を表示する
