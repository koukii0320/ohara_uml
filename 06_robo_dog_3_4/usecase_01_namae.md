# ユースケース名

名前変更

# ユースケース概要

RoboDog の名前を変更する

# アクター

ユーザー

# 事前条件

- メニュー画面が表示されている

# 事後条件

- 名前の変更結果を表示する

# 基本フロー

1. ユーザーは、RoboDog システムメニューの「名前の変更」を選択する
2. システムは、RoboDog が持っている古い名前を表示する
3. システムは、「名前の変更画面」を表示する
4. ユーザーは、「名前の変更画面」に新しい名前を入力する
5. システムは、新しい名前を、RoboDog に登録する
6. システムは、名前の変更結果を表示する
